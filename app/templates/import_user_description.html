Schüler können aus einer CSV-Datei importiert werden, die mithilfe eines Texteditors (ISO-8859-1 Encoding) oder eines Tabellenkalukationsprogramms erstellt wurde. Diese Datei muss die folgenden Spalten enthalten: 

Vorname,Nachname,E-Mail-Adresse,Klasse

Als Vorlage kann diese Beispieldatei verwendet werden. Die erste Zeile mit den Spaltenbezeichnungen muss erhalten bleiben, die zweite Zeile mit dem Beispieldatensatz muss gelöscht werden. Anschließend können weitere Datensätze von Schülern als Zeilen eingefügt werden. Schließlich kann die erstellte CSV-Datei mit dem folgenden Formular hochgeladen werden. Die Lernenden erhalten eine E-Mail mit dem Aktivierungslink, über den ein Passwort gesetzt werden kann. Falls diese nicht im Posteingang gefunden wird, bitte auch den Spam-Ordner kontrollieren.

Sie können Schüler aus einer CSV-Datei, die mithilfe eines Texteditors oder eines Tabellenkalukationsprogramms erstellt wurde, importieren. Diese Datei muss die folgenden Spalten enthalten: 

Vorname,Nachname,E-Mail-Adresse,Klasse

Als Vorlage kann diese Beispieldatei verwendet werden. Die erste Zeile mit den Spaltenbezeichnungen muss erhalten bleiben, die zweite Zeile mit dem Beispieldatensatz muss gelöscht werden. Anschließend können weitere Datensätze von Schülern als Zeilen eingefügt werden.
